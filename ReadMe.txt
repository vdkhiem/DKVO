1. How to setup local database
1.1 Create Empty database in SQL Server with name is "dkvodb"
1.2 Execute the dkvodb.sql script to create tables and procedure

2. Ensure internet connection when open the Solution so VS can restore NuGet Packages
3. Update ConnectionString in DkvoAngularJs/Web.config, DkvoApi/Web.config, DkvoUnitTest/App.config, BOL/App.config
4. Run DkvoAngularJs web application and select Person on top menu
